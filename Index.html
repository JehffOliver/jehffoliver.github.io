Agenda de Contatos

Este projeto é uma aplicação web desenvolvida em C# utilizando o framework .NET Core com a arquitetura MVC (Model-View-Controller). A principal finalidade deste projeto é permitir que os usuários possam gerenciar seus contatos de forma eficiente.

Recursos Principais:

Cadastro de Contatos: Os usuários podem adicionar informações sobre seus contatos, como nome, telefone, e-mail e imagem de perfil.

Edição de Contatos: É possível editar as informações de contato existentes, incluindo a troca da imagem de perfil.

Visualização de Contatos: A aplicação permite que os usuários vejam uma lista de todos os contatos cadastrados.

Remoção de Contatos: Os contatos podem ser excluídos da lista quando necessário.

Prevenção contra SQL Injection: O projeto incorpora medidas de segurança para evitar ataques de SQL Injection, protegendo os dados armazenados.

Armazenamento de Imagens: As imagens de perfil dos contatos são armazenadas como arrays de bytes no banco de dados, garantindo que elas estejam disponíveis para visualização e edição.

Migrações de Banco de Dados: O Entity Framework Core é utilizado para gerenciar o esquema do banco de dados, permitindo migrações automáticas para manter o banco de dados atualizado com as mudanças no modelo de dados.
